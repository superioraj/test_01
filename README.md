# 🐍 Snake Game (HTML / CSS / JavaScript — Single File)

**화살표 키로 조작하는 클래식 스네이크 게임**입니다.  
HTML, CSS, JavaScript가 모두 **하나의 HTML 파일** 안에 포함되어 있어 바로 실행할 수 있습니다.

## 🎮 Features

- 화살표 키(↑ ↓ ← →)로 스네이크 조작  
- 먹이를 먹으면 길이 증가  
- 벽 또는 몸 충돌 시 **게임 오버**  
- 점수 표시  
- 5점마다 자동으로 **속도 증가(가속)**  
- 모든 코드가 **단일 HTML 파일 하나**에 포함  
- Enter 키로 즉시 재시작  
- 그리드 기반 다크 UI + 라운드형 스네이크 디자인

---

## 📝 How to Play

| 조작 | 설명 |
|------|------|
| ↑ ↓ ← → | 방향 이동 |
| 방향키 | 첫 입력 시 게임 시작 |
| Enter | 게임 오버 후 재시작 |
